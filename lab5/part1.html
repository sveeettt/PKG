Алгоритмы отсечения

Средней точки
Кируса-Бека











Используйте колесико мыши на холсте для изменения масштаба
Алгоритм средней точки: Отсекает отрезки прямоугольным окном. Исходные отрезки - красным, видимые части - зеленым.

Особенности алгоритма:

  • Рекурсивно делит отрезок пополам
  • Проверяет видимость средней точки
  • Продолжает для видимых частей
  • Подходит только для прямоугольных окон
  • Сложность: O(log n)
Алгоритм Кируса-Бека: Отсекает отрезки выпуклым многоугольником. Исходные отрезки - красным, видимые части - зеленым.

Особенности алгоритма:

  • Работает с выпуклыми многоугольниками
  • Использует параметрическое представление
  • Вычисляет точки входа и выхода
  • Определяет нормали к ребрам
  • Сложность: O(n) где n - число вершин
