# Алгоритмы отсечения отрезков и многоугольников

Веб-приложение для визуализации алгоритмов отсечения отрезков:  
- **Алгоритм средней точки** (для прямоугольного окна)  
- **Алгоритм Кируса-Бека** (для выпуклого многоугольника)

---

##  Описание проекта

Приложение демонстрирует работу двух классических алгоритмов компьютерной графики:

- **Алгоритм средней точки** — рекурсивное деление отрезка пополам для определения видимых частей в прямоугольном окне.  
- **Алгоритм Кируса-Бека** — параметрическое отсечение отрезков произвольным выпуклым многоугольником.

---

##  Функциональность

### Алгоритм средней точки
- Отсекает отрезки прямоугольным окном  
- Исходные отрезки отображаются **красным цветом**  
- Видимые части — **зелёным цветом**  
- Рекурсивное деление отрезков пополам  

### Алгоритм Кируса-Бека
- Отсекает отрезки выпуклым многоугольником  
- Использует параметрическое представление отрезков  
- Вычисляет точки входа и выхода относительно многоугольника  

---

##  Интерфейс

- Переключение между алгоритмами через вкладки  
- Поля ввода для задания параметров:
  - Координаты отрезков  
  - Параметры окна отсечения (для средней точки)  
  - Вершины многоугольника (для Кируса-Бека)  
- Регулировка масштаба (ползунок и колесико мыши)  
- Кнопки управления: **Отобразить** и **Очистить**  

---

##  Технические особенности

### Алгоритм средней точки
- **Сложность**: `O(log n)`  
- **Ограничение**: только прямоугольные окна  
- **Принцип**: проверка видимости средней точки  

### Алгоритм Кируса-Бека
- **Сложность**: `O(n)`, где `n` — число вершин многоугольника  
- **Ограничение**: многоугольник должен быть выпуклым  
- **Принцип**: вычисление параметров входа/выхода через нормали к рёбрам  

---
