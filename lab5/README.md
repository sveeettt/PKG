# Алгоритмы отсечения отрезков и многоугольников

Визуализатор алгоритмов отсечения отрезков: алгоритма средней точки и алгоритма Кируса-Бека.

## Описание проекта

Проект представляет собой веб-приложение для визуализации двух алгоритмов компьютерной графики:
- **Алгоритм средней точки** - для отсечения отрезков прямоугольным окном
- **Алгоритм Кируса-Бека** - для отсечения отрезков выпуклым многоугольником

## Функциональность

### Алгоритм средней точки
- Отсекает отрезки прямоугольным окном
- Исходные отрезки отображаются красным цветом
- Видимые части отрезков - зеленым цветом
- Рекурсивно делит отрезки пополам для определения видимых частей

### Алгоритм Кируса-Бека
- Отсекает отрезки произвольным выпуклым многоугольником
- Использует параметрическое представление отрезков
- Вычисляет точки входа и выхода отрезка относительно многоугольника

## Интерфейс

Приложение предоставляет:
- Переключение между алгоритмами через вкладки
- Поля ввода для задания параметров:
  - Координаты отрезков
  - Параметры окна отсечения (для алгоритма средней точки)
  - Вершины многоугольника (для алгоритма Кируса-Бека)
- Регулировка масштаба (ползунок и колесико мыши)
- Кнопки управления: отображение и очистка

## Технические особенности

### Алгоритм средней точки
- **Сложность**: O(log n)
- **Ограничение**: работает только с прямоугольными окнами
- **Принцип**: рекурсивное деление отрезка пополам и проверка видимости средней точки

### Алгоритм Кируса-Бека
- **Сложность**: O(n), где n - количество вершин многоугольника
- **Ограничение**: требует выпуклый многоугольник
- **Принцип**: вычисление параметров входа и выхода с использованием нормалей к ребрам

## Используемые технологии

- HTML5 Canvas для отрисовки
- Pure JavaScript для реализации алгоритмов
- CSS для стилизации интерфейса
- Адаптивный дизайн

##Управление
Изменение масштаба: колесико мыши на холсте или поле ввода "Масштаб"

Переключение алгоритмов: вкладки "Средней точки" и "Кируса-Бека"

Отображение: кнопка "Отобразить" - применяет выбранный алгоритм

Очистка: кнопка "Очистить" - очищает холст и поля ввода
