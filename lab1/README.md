Конвертер цветовых моделей HSV ↔ XYZ ↔ LAB

Назначение программы
Данное приложение представляет собой интерактивный конвертер цветов между тремя цветовыми моделями: HSV, XYZ и LAB. Программа позволяет пользователю задавать цвет различными способами и автоматически пересчитывает значения между всеми моделями при любом изменении.

Основные возможности

Три способа задания цвета для каждой модели:
1.Точный числовой ввод - через поля ввода 
2.Плавное изменение - с помощью ползунков
3.Визуальный выбор - через стандартную палитру цветов

Поддерживаемые цветовые модели:
HSV (Hue, Saturation, Value):
H: 0-360° (оттенок)
S: 0-100% (насыщенность)
V: 0-100% (яркость)

XYZ (CIE 1931):
X: 0-100
Y: 0-100
Z: 0-100

LAB (CIE L*a*b*):
L*: 0-100 (яркость)
a*: -128-128 (зелено-красная ось)
b*: -128-128 (сине-желтая ось)

Установка

Требования:
Qt 5.15 или выше
Компилятор с поддержкой C++11

Сборка из исходного кода:
Склонируйте репозиторий на локальный компьютер при помощи
git clone <URL_репозитория>

Убедитесь, что у вас установлен Qt (минимум версия Qt 5)

Откройте проект в Qt Creator или аналогичной среде

Соберите и запустите проект

В главном окне программы представлены:
Визуальное отображение текущего цвета - показывает выбранный цвет
Кнопка "Выбрать из палитры" - открывает стандартный диалог выбора цвета
Три группы элементов управления для каждой цветовой модели:
Текстовые поля для точного ввода значений
Ползунки для плавного изменения значений
Подписи с указанием диапазонов допустимых значений

Управление ползунками и текстовыми полями:
Значения в текстовых полях ограничиваются заданными диапазонами:

HSV: H(0-360), S(0-100), V(0-100)

XYZ: X(0-100), Y(0-100), Z(0-100)

LAB: L(0-100), a(-128-128), b(-128-128)

При изменении значения ползунка соответствующее текстовое поле обновляется автоматически

При вводе значений в текстовые поля программа автоматически обновляет все другие модели

Функции и методы

Основные функции:
updateFromHSV(), updateFromXYZ(), updateFromLAB() - обработчики изменений значений в соответствующих моделях

openColorPalette() - открывает диалог выбора цвета из палитры

updateFromHSVSliders(), updateFromXYZSliders(), updateFromLABSliders() - обработчики изменений ползунков

Вспомогательные функции:

updateAllFromHSV(), updateAllFromXYZ(), updateAllFromLAB() - пересчитывают все модели на основе изменений в одной из них
updateColorDisplay() - обновляет визуальное отображение цвета
showWarning() - отображает сообщения об ошибках

Класс ColorTransformer:

hsvToXyz(), xyzToHsv() - преобразования между HSV и XYZ
xyzToLab(), labToXyz() - преобразования между XYZ и LAB
labF(), labFInv() - вспомогательные функции для преобразований LAB

Ошибки и предупреждения

Программа включает проверку корректности значений:
Все поля имеют установленные диапазоны значений
При преобразованиях выполняется проверка на выход за границы допустимых значений
Реализована система предупреждений о некорректных значениях

Пример использования
Запустите программу - по умолчанию установлен серый цвет (H=0, S=0, V=50)
Измените цвет любым способом:
Введите значения в поля HSV, XYZ или LAB
Используйте ползунки для плавного изменения
Нажмите "Выбрать из палитры" для визуального выбора

Наблюдайте автоматическое обновление:
При изменении любой компоненты все три модели автоматически пересчитываются
Визуальное отображение цвета обновляется в реальном времени


Заключение
Данный проект предоставляет удобный и интуитивно понятный интерфейс для работы с различными цветовыми моделями.
Он позволяет легко конвертировать цвета между моделями HSV, XYZ и LAB, имеет встроенную проверку на допустимые диапазоны 
значений и предоставляет multiple способы взаимодействия с цветом. Программа будет полезна дизайнерам, разработчикам и всем,
кто работает с цветом в профессиональной деятельности.
