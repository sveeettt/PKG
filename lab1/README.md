# Конвертер цветовых моделей HSV ↔ XYZ ↔ LAB

## Назначение программы

Интерактивный конвертер цветов между тремя цветовыми моделями: **HSV**, **XYZ** и **LAB**. Позволяет задавать цвет различными способами и автоматически пересчитывает значения между всеми моделями при любом изменении.

## Основные возможности

### Три способа задания цвета для каждой модели:
1. **Точный числовой ввод** - через поля ввода
2. **Плавное изменение** - с помощью ползунков  
3. **Визуальный выбор** - через стандартную палитру цветов

### Поддерживаемые цветовые модели:

#### HSV (Hue, Saturation, Value)
- **H**: 0-360° (оттенок)
- **S**: 0-100% (насыщенность) 
- **V**: 0-100% (яркость)

#### XYZ (CIE 1931)
- **X**: 0-100
- **Y**: 0-100
- **Z**: 0-100

#### LAB (CIE Lab*)
- **L***: 0-100 (яркость)
- **a***: -128-128 (зелено-красная ось)
- **b***: -128-128 (сине-желтая ось)

## Интерфейс программы

В главном окне представлены:

- **Визуальное отображение текущего цвета** - показывает выбранный цвет
- **Кнопка "Выбрать из палитры"** - открывает стандартный диалог выбора цвета
- **Три группы элементов управления** для каждой цветовой модели:
  - Текстовые поля для точного ввода значений
  - Ползунки для плавного изменения значений
  - Подписи с указанием диапазонов допустимых значений

### Управление элементами:

- Значения в текстовых полях ограничиваются заданными диапазонами
- При изменении ползунка соответствующее текстовое поле обновляется автоматически
- При вводе значений в текстовые поля программа автоматически обновляет все другие модели

## Функциональность

### Основные функции:
- `updateFromHSV()`, `updateFromXYZ()`, `updateFromLAB()` - обработчики изменений значений
- `openColorPalette()` - открывает диалог выбора цвета из палитры
- `updateFromHSVSliders()`, `updateFromXYZSliders()`, `updateFromLABSliders()` - обработчики ползунков

### Вспомогательные функции:
- `updateAllFromHSV()`, `updateAllFromXYZ()`, `updateAllFromLAB()` - пересчет всех моделей
- `updateColorDisplay()` - обновление визуального отображения цвета
- `showWarning()` - отображение сообщений об ошибках

### Класс ColorTransformer:
- `hsvToXyz()`, `xyzToHsv()` - преобразования между HSV и XYZ
- `xyzToLab()`, `labToXyz()` - преобразования между XYZ и LAB
- `labF()`, `labFInv()` - вспомогательные функции для преобразований LAB

## Обработка ошибок

Программа включает проверку корректности значений:

- Все поля имеют установленные диапазоны значений
- При преобразованиях выполняется проверка на выход за границы допустимых значений
- Реализована система предупреждений о некорректных значениях

## Пример использования

1. **Запустите программу** - по умолчанию установлен серый цвет (H=0, S=0, V=50)
2. **Измените цвет любым способом**:
   - Введите значения в поля HSV, XYZ или LAB
   - Используйте ползунки для плавного изменения
   - Нажмите "Выбрать из палитры" для визуального выбора
3. **Наблюдайте автоматическое обновление**:
   - При изменении любой компоненты все три модели автоматически пересчитываются
   - Визуальное отображение цвета обновляется в реальном времени

## Заключение

Проект предоставляет удобный и интуитивно понятный интерфейс для работы с различными цветовыми моделями. Позволяет легко конвертировать цвета между моделями HSV, XYZ и LAB, имеет встроенную проверку на допустимые диапазоны значений и предоставляет multiple способы взаимодействия с цветом.
